Wishlist

Track Bands You Want to See

The Wishlist helps you keep track of bands you haven't seen yet but want to.

Adding to Wishlist

From Band Detail

  1. Open any band
  2. Tap the Wishlist button (star icon)
  3. Band is now on your wishlist

Quick Add

  1. Go to More → Wishlist
  2. Tap the + button
  3. Enter band name
  4. Optionally set genre and country
  5. Tap Add

From Lineup Checker

  1. Paste a festival lineup
  2. Check the lineup
  3. Select "New" bands
  4. Tap Add to Wishlist

From Band Origins Missing

  1. Go to Maps → Band Origins
  2. Select Missing tab
  3. Tap a country for suggestions
  4. Tap + to add bands to wishlist

Wishlist Views

The wishlist screen has multiple tabs:

Bands Tab

  • All wishlist bands
  • Shows if band exists in your collection
  • Smart matching links wishlist to seen bands

Events Tab

  • Events featuring wishlist bands
  • Useful for planning what shows to attend

Venues Tab

  • Venues where wishlist bands have played
  • Helps discover new venues

Smart Matching

When you see a wishlist band live:

  1. Add the band to an event
  2. PocketDemon detects the match
  3. Wishlist entry can be automatically removed
  4. Or keep it for "want to see again"

MusicBrainz Enrichment

Create full band records from wishlist entries:

  1. Open a wishlist band
  2. Tap Create Band
  3. PocketDemon searches MusicBrainz
  4. Select the correct match
  5. Band is created with full metadata

Ticketmaster Scanner

Find concerts for wishlist bands:

  1. Go to More → Event Scanner
  2. Select Wishlist as source
  3. Set search radius and location
  4. Tap Scan
  5. See upcoming shows for your wishlist bands

Note

Ticketmaster scanner has a daily rate limit of 1000 requests. Use wisely!

Removing from Wishlist

  1. Open the wishlist band
  2. Tap the Wishlist button again
  3. Or swipe to delete in the list

Wishlist Statistics

Track your wishlist progress:

  • Total wishlist bands
  • Completed (bands now seen)
  • Completion rate
  • Time to see (average wait)
