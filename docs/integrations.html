Integrations

Connect PocketDemon with your favorite music services

Available Integrations

PocketDemon connects with various music services to enhance your experience and import data.

Integrations settings screen

Setlist.fm

The premier source for concert setlists worldwide.

Features

  • View setlists - See what songs were played at concerts you attended
  • Import concerts - Pull your attended concerts from your setlist.fm profile
  • Song statistics - See your most heard songs, rare songs, and covers from cached setlists
  • Genius lyrics - Tap any song in a setlist to search for lyrics on Genius

Usage

  1. Open any concert in your collection
  2. Tap View Setlist
  3. PocketDemon fetches the setlist from setlist.fm
  4. Tap any song to view lyrics on Genius

Import from Setlist.fm

  1. Go to Settings > Import > Setlist.fm
  2. Enter your setlist.fm username
  3. Tap Import to fetch your attended concerts

Note

Setlists require an internet connection and may not be available for all concerts. Song statistics are calculated from setlists you've viewed (cached locally).

Last.fm

Import your listening history and top artists.

Features

  • Top artists - Import your most-played artists to your wishlist
  • Library artists - Import all scrobbled artists with play counts
  • Play counts - See how many times you've listened to each artist

Setup

  1. Go to Settings > Import > Last.fm
  2. Enter your Last.fm username
  3. Tap Import Top Artists or Import Library

Note

Last.fm import uses read-only access via username - no OAuth required. Requires a Last.fm API key configured in the app build.

Discogs

Import your physical music collection.

Features

  • Collection import - Import your vinyl, CD, and cassette collection
  • Release details - Artist, year, label, catalog number, format
  • Band linking - Link imported releases to bands in your concert collection

Setup

  1. Go to More > Collection > Import
  2. Enter your Discogs username
  3. Tap Import Collection
  4. Review and confirm the import

Spotify

Artist search and band linking.

On Hold

Spotify integration is currently on hold due to Spotify API access restrictions implemented in late 2024. The code is ready for when access becomes available again.

Planned Features

  • Artist search - Search and link Spotify artists to bands
  • Spotify links - Open bands directly in Spotify
  • Import followed artists - Add followed artists to wishlist
  • Setlist playlists - Create playlists from concert setlists

Dropbox

Cloud storage for backups, photo sync, and buddy connections.

Features

  • Full backup - All your data synced to the cloud
  • Photo sync - Upload and restore concert photos
  • Auto-sync - Battery-friendly sync on app pause (15-minute cooldown)
  • Buddy connections - Stable buddy identity via Dropbox account
  • Cross-device sync - Access your data on multiple devices

Setup

  1. Go to Settings > Backup > Dropbox
  2. Tap Connect
  3. Authorize PocketDemon in Dropbox
  4. Sync happens automatically when the app is backgrounded

Desktop OAuth

On Windows, macOS, and Linux, Dropbox OAuth uses a localhost HTTP server for the redirect. A browser window will open for authorization.

MusicBrainz

Open music encyclopedia for comprehensive band data.

Features

  • Band enrichment - Formation year, country, genres, status
  • Aliases - Alternative names and spellings
  • External links - Wikipedia, Discogs, AllMusic, Last.fm
  • Social links - Spotify, YouTube, Facebook, Twitter, Instagram

Usage

  1. Open any band's detail page
  2. Tap Enrich (or menu > Enrich from MusicBrainz)
  3. Data is fetched automatically - no account needed

Rate Limiting

MusicBrainz limits requests to 1 per second. Enrichment may take a moment if you're enriching multiple bands.

TheAudioDB

Artist photos, logos, and biographies.

Features

  • Artist photos - High-quality artist images
  • Band logos - Official logo images
  • Biography - Artist biography text
  • Style - Sub-genre classification

Usage

TheAudioDB is queried automatically when enriching bands. Photos are downloaded and stored locally for offline access.

Ticketmaster

Venue data and nearby concert discovery.

Features

  • Venue enrichment - Photos, addresses, and GPS coordinates for venues
  • Event scanner - Discover nearby concerts for bands in your Wishlist or Favorites
  • Radius search - Configure search radius and city

Event Scanner

  1. Go to Events > Scanner
  2. Set your location and search radius
  3. Tap Scan to find upcoming concerts
  4. Results show bands from your Wishlist and Favorites

Rate Limiting

Ticketmaster API has a daily rate limit of 1000 requests. The scanner tracks usage to prevent exceeding this limit.

Genius

Song lyrics via browser search.

Features

  • Lyrics search - Quick links to song lyrics from setlist view

Usage

When viewing a setlist, tap the lyrics icon next to any song. This opens Genius search in your browser - no API key required.

Calendar Export (ICS)

Export events to your calendar app.

Features

  • ICS export - Standard calendar format supported by all calendar apps
  • Multi-day support - Festivals exported as all-day events
  • Event details - Includes venue, bands, and notes

Usage

  1. Open any event
  2. Tap menu > Export to Calendar
  3. An ICS file is generated and shared
  4. Open with iOS Calendar, Google Calendar, Outlook, etc.

Managing Integrations

Disconnect an Integration

  1. Go to Settings > Integrations
  2. Find the connected service
  3. Tap Disconnect
  4. Confirm disconnection

Data Privacy

  • PocketDemon only requests necessary permissions
  • Your credentials are never stored - only OAuth tokens
  • You can revoke access anytime from the service's settings
  • Disconnecting removes all stored tokens

Disconnecting Services

Disconnecting an integration won't delete data already imported. To remove imported data, use the respective data management options.
