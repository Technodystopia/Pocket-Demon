Integrations

Connect PocketDemon with your favorite music services

Available Integrations

PocketDemon connects with various music services to enhance your experience and import data.

Integrations settings screen

Setlist.fm

The premier source for concert setlists worldwide.

Features

  • Import concerts - Pull your concert history from setlist.fm
  • Setlist lookup - Find setlists for concerts you've attended
  • Attach setlists - Link setlist.fm data to your concert entries
  • Submit setlists - Contribute setlists back to the community

Setup

  1. Go to Settings > Integrations > Setlist.fm
  2. Enter your setlist.fm username
  3. Optionally connect your API key for enhanced features

API Key

For full integration features, you can get a free API key at setlist.fm/settings/api

Last.fm

Connect your scrobbling history for automatic concert matching.

Features

  • Listening history - See what you listened to before/after shows
  • Band discovery - Find bands you listen to but haven't seen live
  • Scrobble data - Import listening stats for bands
  • Concert suggestions - Get recommendations based on listening habits

Setup

  1. Go to Settings > Integrations > Last.fm
  2. Tap Connect Account
  3. Authorize PocketDemon in the Last.fm popup
  4. Your accounts are now linked

Import Options

  • Concert history - Events marked as attended
  • Top artists - Your most-played bands
  • Recent plays - What you've been listening to

Discogs

For physical music collectors.

Features

  • Collection sync - Import/export your vinyl and CD collection
  • Wantlist sync - Keep your wishlist in sync
  • Barcode lookup - Identify releases by scanning
  • Value tracking - Current market values for your collection

Setup

  1. Go to Settings > Integrations > Discogs
  2. Tap Connect Account
  3. Log in to Discogs and authorize PocketDemon
  4. Choose sync preferences:
    • One-time import
    • Continuous sync (both directions)
    • PocketDemon as primary (push only)

Spotify

Connect with the world's most popular streaming service.

Features

  • Pre-show playlists - Auto-generate playlists for upcoming concerts
  • Band lookup - Enrich band data with Spotify info
  • Popularity data - See how popular bands are
  • Direct playback - Open bands directly in Spotify

Setup

  1. Go to Settings > Integrations > Spotify
  2. Tap Connect Account
  3. Log in to Spotify and authorize PocketDemon

Pre-Show Prep

Before a concert, use the "Create Playlist" feature to make a Spotify playlist of the expected setlist based on recent shows!

Dropbox

Cloud storage for backups and photo sync.

Features

  • Automatic backups - Your data saved to the cloud
  • Photo backup - Concert photos stored safely
  • Cross-device sync - Access your data on multiple devices
  • Version history - Restore previous backup versions

Setup

  1. Go to Settings > Integrations > Dropbox
  2. Tap Connect Account
  3. Authorize PocketDemon in Dropbox
  4. Configure backup frequency:
    • Real-time
    • Daily
    • Weekly
    • Manual only

MusicBrainz

Open music encyclopedia for comprehensive band data.

Features

  • Band enrichment - Detailed band information
  • Discography data - Complete release histories
  • Band relationships - Members, side projects
  • Genre classification - Accurate genre tags

Usage

MusicBrainz integration is automatic - no account needed. PocketDemon queries MusicBrainz when enriching band data.

Bandcamp

Support artists directly with Bandcamp integration.

Features

  • Purchase history - Import your Bandcamp purchases
  • Collection sync - Add Bandcamp purchases to your collection
  • Direct links - Open bands on Bandcamp to support them
  • Wishlist - Sync your Bandcamp wishlist

Calendar Integration

Keep your concerts in your calendar app.

Features

  • Export to calendar - Add concerts as calendar events
  • Automatic updates - Changes sync to your calendar
  • Reminders - Get notified before shows

Supported Calendars

  • Google Calendar
  • Apple Calendar (iOS)
  • Outlook Calendar
  • Any CalDAV-compatible calendar

Managing Integrations

Disconnect an Integration

  1. Go to Settings > Integrations
  2. Find the connected service
  3. Tap Disconnect
  4. Confirm disconnection

Data Privacy

  • PocketDemon only requests necessary permissions
  • Your credentials are never stored - only OAuth tokens
  • You can revoke access anytime from the service's settings
  • Disconnecting removes all stored tokens

Disconnecting Services

Disconnecting an integration won't delete data already imported. To remove imported data, use the respective data management options.
