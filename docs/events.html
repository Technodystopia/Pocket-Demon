Events & Concerts

Understanding Events vs Concerts

In PocketDemon:

  • Event = A show at a venue on a specific date (e.g., "Wacken Open Air 2024")
  • Concert = A band performing at an event (e.g., "Slayer at Wacken 2024")

One event can have multiple concerts (all the bands you saw that day/night).

Creating an Event

  1. Go to the Events tab
  2. Tap the + button
  3. Fill in the details:
    • Name - Event or tour name
    • Venue - Select existing or create new
    • Date - When it happened
    • End Date - For multi-day festivals
    • Is Festival - Toggle for festival events
    • Ticket Price - Optional cost tracking
    • Currency - 13+ currencies supported
    • Notes - Personal memories
  4. Tap Save

Event Types

Single Shows

A regular concert on one evening. Just set the date and leave end date empty.

Multi-Day Events

Festivals spanning multiple days:

  1. Set the start date
  2. Set the end date
  3. Enable Is Festival

The app will show the full date range and track all bands across all days.

Cruise Ship Events

For concerts on cruise ships (like 70000 Tons of Metal):

  1. Create a venue marked as "Cruise Ship"
  2. Create the event at that venue
  3. Add port stops with dates

Cruise events support full itinerary tracking with multiple ports.

Adding Bands to Events

  1. Open the event
  2. Tap Add Band
  3. Search for the band or create new
  4. Set your rating (1-5 stars)
  5. Add optional notes about the performance
  6. Tap Add

Tip

You can add the same band multiple times to one event (e.g., if they played two sets at a festival).

Concert Ratings

Rate each performance from 1-5 stars. This affects:

  • Average rating calculations in statistics
  • Year in Review highlights
  • Sorting and filtering options

Concert Reviews

For more detailed feedback, use the multi-criteria review system:

  1. Open a concert (band at event)
  2. Tap Add Review
  3. Rate five categories:
    • Performance
    • Sound Quality
    • Stage Presence
    • Setlist
    • Overall Experience
  4. Add a written review (optional)
  5. Save the review

Ticket Tracking

Track your concert spending:

  • Price - What you paid
  • Currency - EUR, USD, GBP, SEK, and 10+ more
  • Ticket Type - General, VIP, Weekend Pass, etc.

View spending analysis in the Spending Tracker.

Event Photos

Add photos to remember the night:

  1. Open the event
  2. Tap Add Photo
  3. Select from gallery or take new
  4. Photos are automatically compressed to save space

See Photo Gallery for more on managing photos.

Setlists

View what songs were played:

  1. Open a concert
  2. Tap View Setlist
  3. PocketDemon fetches the setlist from setlist.fm
  4. Tap any song for Genius lyrics

Note

Setlists require an internet connection and may not be available for all concerts.

Concert Buddies

Track who you went with:

  1. Open the event
  2. Tap Add Buddy
  3. Select from your buddies list

See Concert Buddies for setting up buddies.

Timeline View

Browse your concert history chronologically:

  1. Go to More → Timeline
  2. Events are grouped by Year → Month → Event
  3. Tap to expand and see bands at each event

Calendar Export

Export events to your calendar app:

  1. Open the event
  2. Tap menu → Export to Calendar
  3. An ICS file is created
  4. Open with iOS Calendar, Google Calendar, etc.

Future Events

You can add future events to plan your concert calendar. Note:

  • Future events appear in the Events list
  • They are excluded from statistics
  • They appear on the calendar view
  • Once the date passes, they count toward stats
