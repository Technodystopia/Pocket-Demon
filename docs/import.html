Import Data

Bring Your Concert History

PocketDemon can import your concert history from several sources.

Setlist.fm Import

Import concerts you've marked as "attended" on setlist.fm:

  1. Go to Settings → Import
  2. Tap Import from Setlist.fm
  3. Enter your setlist.fm username
  4. Tap Search
  5. Review found concerts
  6. Select which to import
  7. Tap Import Selected

What Gets Imported

  • Event name and date
  • Venue with city and country
  • Band name
  • Setlist (cached for later viewing)

Tip

After importing, use the Enrich feature on bands to fetch additional metadata from MusicBrainz.

Last.fm Import

Import artists from your Last.fm library:

Top Artists

  1. Go to Settings → Import
  2. Tap Import from Last.fm
  3. Enter your Last.fm username
  4. Select Top Artists
  5. Choose time period (overall, year, month, week)
  6. Review the list
  7. Import as bands or add to wishlist

Library Artists

  1. Same steps, but select Library Artists
  2. Shows all artists in your Last.fm library
  3. Includes play counts

Note

Last.fm import adds bands to your collection but doesn't create events - you'll need to add the concert details manually.

Discogs Import

Import your physical media collection from Discogs:

  1. Go to More → Collection
  2. Tap menu → Import from Discogs
  3. Enter your Discogs username
  4. Select collection folder
  5. Review releases
  6. Import to your collection

What Gets Imported

  • Artist/band name
  • Release title
  • Year
  • Label and catalog number
  • Format (vinyl, CD, cassette)
  • Format details (LP, 7", etc.)

CSV Import

Import from spreadsheets:

  1. Go to Settings → Import
  2. Tap Import CSV
  3. Select your CSV file
  4. Map columns to fields
  5. Preview import
  6. Confirm import

CSV Format

Required columns depend on what you're importing:

Bands CSV

name,country,genres
Metallica,United States,"Thrash Metal, Heavy Metal"
Iron Maiden,United Kingdom,Heavy Metal

Events CSV

name,date,venue,city,country
Wacken 2024,2024-08-01,Wacken Festival Grounds,Wacken,Germany

Duplicate Detection

PocketDemon detects duplicates during import:

  • Exact name matches are flagged
  • Similar names are suggested for review
  • Choose to skip, merge, or create new

Lineup Checker Import

Quick way to add bands from festival lineups:

  1. Go to More → Lineup Checker
  2. Paste the lineup (any format)
  3. Tap Check Lineup
  4. See which bands you've seen, wishlisted, or are new
  5. Select new bands to add to wishlist

Supported Formats

The parser handles various lineup formats:

  • One band per line
  • Comma-separated
  • Bullet points
  • With or without times/stages

Import Best Practices

  1. Backup first - Always backup before large imports
  2. Start small - Test with a few records first
  3. Review duplicates - Take time to resolve matches
  4. Enrich after - Use batch enrich for imported bands
  5. Check achievements - New badges may unlock!
