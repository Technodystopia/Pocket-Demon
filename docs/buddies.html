Concert Buddies

Track Who You Go With

Concert Buddies lets you track friends you attend concerts with, compare stats, and share data.

Adding a Buddy

There are several ways to add buddies:

QR Code (Recommended)

  1. Go to More → Buddies
  2. Tap My QR Code to show yours
  3. Have your friend scan it, or scan theirs
  4. Both users are now connected

Manual Entry

  1. Go to Buddies
  2. Tap the + button
  3. Enter buddy's name
  4. Tap Save

Note

Manually added buddies are local only. Use QR codes or Dropbox connections for full buddy features.

Dropbox Buddy Connections

For the best buddy experience, connect via Dropbox:

  1. Both users must have Dropbox sync enabled
  2. Scan each other's QR code (v2 format includes Dropbox ID)
  3. Send a connection request
  4. Accept the request when received

Benefits of Dropbox connections:

  • Stable identity across devices
  • Automatic profile sync
  • Share bands, events, and venues
  • Real-time updates

Sharing Levels

Control what each buddy can see:

Level What They See
Stats Only Total counts, top bands, countries - no event details
Events Stats plus event names and dates
Full Access Complete access to bands, events, and venues

Setting Sharing Level

  1. Open the buddy's profile
  2. Tap Sharing Settings
  3. Select the sharing level
  4. Changes sync automatically

Attaching Buddies to Events

  1. Open an event
  2. Tap Add Buddy
  3. Select buddies who attended
  4. They'll appear on the event detail

Concert Twin Score

See how compatible you are with each buddy:

  • Shared Bands - Bands you've both seen
  • Shared Events - Events attended together
  • Genre Overlap - Similar music taste
  • Twin Score - Overall compatibility percentage

Buddy Leaderboard

The leaderboard shows your top 3 compatible buddies on a podium:

  1. Go to Buddies
  2. Tap Leaderboard
  3. See podium with gold, silver, bronze positions
  4. Based on concert twin score

Group Stats

Compare statistics across your buddy group:

  • Combined Stats - Total across all buddies
  • Overlap Metrics - What you share in common
  • Gap Analysis - Bands they've seen that you haven't
  • Trend Comparison - Who's been more active

Entity Sharing

Share specific items with buddies:

Sharing a Band

  1. Open the band detail
  2. Tap Share → Share with Buddy
  3. Select which buddies to share with
  4. They'll receive it in their inbox

Buddy Inbox

Receive shared items from buddies:

  1. Go to Buddies → Inbox
  2. See pending shared items
  3. Tap to preview
  4. Accept to add to your collection
  5. Dismiss to ignore

Bluetooth Sharing

Share at festivals without WiFi:

  1. Go to Buddies → BLE Share
  2. One person taps Advertise
  3. Other person taps Scan
  4. Select what to share
  5. Data transfers directly via Bluetooth

Platform Note

Bluetooth sharing is only available on Android and iOS. Not supported on desktop platforms.

Concert Attendance Proof

Create shareable social media cards:

  1. Open an event
  2. Tap Share → Attendance Proof
  3. A card is generated with event details
  4. Share to Instagram, Twitter, etc.
