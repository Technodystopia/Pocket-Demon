Settings

Customize PocketDemon to fit your preferences

Accessing Settings

Tap the gear icon in the top right corner from any main screen, or navigate to Settings from the main menu.

Account

Profile

  • Display name - How you appear to buddies
  • Username - Your unique identifier
  • Profile photo - Your avatar image
  • Bio - Short description about you

Privacy

  • Profile visibility - Public, Friends only, Private
  • Concert history - Who can see your past shows
  • Statistics - Who can view your stats
  • Location sharing - Enable/disable location features

Appearance

Theme

  • Dark mode - Always dark (default)
  • Monochrome mode - OLED battery-saving true black

Accent Colors

Choose from 16 preset accent colors or create your own:

  • Preset colors - 16 carefully selected colors
  • Custom color picker - Full color wheel for primary and secondary accent colors
  • Background colors - Customize scaffold, surface, and card colors

Theme Presets

Share your theme with others or import theirs:

  • Export theme - Generate a shareable theme code (format: PD1:xxxxxxxx:xxxxxxxx)
  • Import theme - Paste a theme code to apply it

Haptic Feedback

  • Enable/disable - Toggle tactile feedback on interactions

Gallery Layout

  • Photos per row - Configure 1-5 photos per row in main gallery
  • Band gallery - Configure photos per row (2-6) and max rows (1-5) for band detail gallery

Quick Actions

  • Customize - Reorder and show/hide home screen quick action buttons
  • Edit mode - Long press to enter edit mode and rearrange buttons

Preferences

Language

PocketDemon supports 12 languages (all fully translated with 900+ strings):

  • English (source)
  • German (Deutsch)
  • Swedish (Svenska)
  • Norwegian (Norsk)
  • Finnish (Suomi)
  • Estonian (Eesti)
  • Spanish (Espanol)
  • Portuguese (Portugues)
  • French (Francais)
  • Dutch (Nederlands)
  • Polish (Polski)
  • Italian (Italiano)

The app automatically uses your device language if supported, or you can set it manually.

Date Display

  • Locale-aware formatting - Dates formatted according to your language setting
  • Birthday - Set your birthday to see your age at each concert year in timeline view

Currency

  • Preferred currency - Default currency for spending displays (EUR, USD, GBP, SEK, and 10+ more)
  • Per-tab selector - Some spending tabs let you choose which currency to display

Home Location

  • Set location - Configure your home city for accurate distance calculations

Venue Display

Venue List Options

  • Country flags - Show colorful or grayscale flags
  • Compact mode - Toggle for denser venue lists

Band Origins Map

  • Display settings - Configure flag display on band origins map

Polaroid Style

Customize how photos appear in polaroid view:

Style Presets

  • Classic (white frame)
  • Dark
  • Vintage (sepia)
  • Minimal
  • Neon

Font Options

  • Handwritten (Caveat)
  • Typewriter (Courier)
  • Modern (sans-serif)
  • Elegant (serif)

Visibility Toggles

  • Show/hide caption
  • Show/hide venue
  • Show/hide date

Per-Photo Settings

Each photo can have its own polaroid style settings, configured when viewing the photo.

Data Management

Backup Options

  • JSON backup - Full backup with validation and undo support
  • JSON restore - Restore from a previous backup file
  • CSV export - Spreadsheet-compatible export
  • CSV import - Import with duplicate detection

Dropbox Sync

  • Connect Dropbox - Enable cloud backup
  • Auto-sync - Battery-friendly sync on app pause (15-minute cooldown)
  • Photo sync - Upload and restore concert photos
  • Sync now - Trigger manual sync

Offline Operation

  • Operation queue - Changes are queued when offline and replayed on reconnect
  • Connectivity indicator - Visual status showing online/offline state

Data Safety

Always create a backup before making major changes. Deleted data cannot be recovered without a backup.

Integrations

Manage your connected services:

  • Dropbox - Cloud backup and buddy connections
  • Setlist.fm - Setlist viewing and concert import
  • Last.fm - Artist import
  • Discogs - Collection import
  • MusicBrainz - Band enrichment (automatic)
  • TheAudioDB - Artist photos (automatic)
  • Ticketmaster - Venue data and event scanner

See the Integrations page for detailed setup instructions.

Maps

Map Provider

  • OpenStreetMap - Default map provider via flutter_map

Offline Map Caching

  • Tile caching - Map tiles are cached for offline viewing
  • Cache management - View and clear cached tiles

Desktop Features

Additional features on Windows, macOS, and Linux:

Keyboard Shortcuts

  • 1-5 - Navigate between main tabs
  • Ctrl+F - Focus search
  • Escape - Go back
  • Ctrl+, - Open settings

Window State

  • Size persistence - Window size remembered across sessions
  • Position persistence - Window position remembered
  • Maximized state - Maximized state remembered

Responsive Layouts

  • Grids adapt to screen width (4/6/8 columns)
  • Content width constraints for comfortable reading

Help & Support

  • User manual - Opens this documentation (first-time users are directed here automatically)
  • Send feedback - Anonymous in-app feedback without email client

First-Time Users

New users are automatically directed to the manual after registration to help them get started.
