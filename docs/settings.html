Settings

Customize PocketDemon to fit your preferences

Accessing Settings

Tap the gear icon in the top right corner from any main screen, or navigate to Settings from the main menu.

Account

Profile

  • Display name - How you appear to buddies
  • Username - Your unique identifier
  • Profile photo - Your avatar image
  • Bio - Short description about you

Privacy

  • Profile visibility - Public, Friends only, Private
  • Concert history - Who can see your past shows
  • Statistics - Who can view your stats
  • Location sharing - Enable/disable location features

Appearance

Theme

  • Dark mode - Dark theme (default)
  • Light mode - Light theme
  • System - Follow device settings
  • OLED Black - True black for OLED screens

Accent Color

Choose your accent color for highlights and buttons:

  • Purple (default)
  • Blue
  • Red
  • Green
  • Orange
  • Custom (pick any color)

Text Size

  • Small
  • Medium (default)
  • Large
  • Extra Large

Preferences

Region & Language

  • Language - App interface language
  • Region - For date formats and defaults
  • Date format - MM/DD/YYYY, DD/MM/YYYY, etc.
  • Time format - 12-hour or 24-hour

Currency

  • Default currency - For spending tracking
  • Show conversions - Display equivalent in other currencies

Units

  • Distance - Miles or kilometers

Notifications

Concert Reminders

  • Upcoming concerts - Days before to remind
  • Day of show - Morning reminder
  • Pre-show - Hours before start time

Social

  • Buddy requests - New connection requests
  • Buddy activity - When buddies attend shows
  • Shared concerts - Mutual show notifications

Achievements

  • Achievement unlocked - Badge notifications
  • Progress updates - Milestone approaching

Wishlist

  • Show alerts - When wishlist bands announce tours
  • Ticket alerts - When tickets go on sale

Data Management

Backup

  • Auto-backup - Enable/disable automatic backups
  • Backup frequency - How often to backup
  • Backup location - Local or cloud
  • Manual backup - Create backup now

Storage

  • Cache size - View and clear cache
  • Photo storage - Local vs cloud photos
  • Optimize storage - Reduce local storage use
  • Offline maps - Download maps for offline use

Export

  • Export all data - Complete JSON backup
  • Export concerts - CSV of concert history
  • Export statistics - Your stats summary

Delete

  • Clear all data - Start fresh (irreversible)
  • Delete account - Remove your account entirely

Data Deletion

Deleting data is permanent and cannot be undone. Always create a backup before deleting anything.

Integrations

Manage your connected services:

  • Setlist.fm - Connect/disconnect
  • Last.fm - Connect/disconnect
  • Discogs - Connect/disconnect
  • Spotify - Connect/disconnect
  • Dropbox - Connect/disconnect

See the Integrations page for detailed setup instructions.

Maps

Map Provider

  • Default - System default
  • OpenStreetMap - Open source maps
  • Google Maps - If available
  • Apple Maps - iOS only

Offline Maps

  • Download regions - Save maps for offline use
  • Auto-download - Cache maps for upcoming concerts
  • Storage limit - Max space for offline maps

Advanced

Developer Options

  • Debug mode - Enable detailed logging
  • API timeout - Adjust connection timeouts
  • Cache lifetime - How long to cache data

Experimental Features

Try new features before they're officially released:

  • Beta features - Enable/disable beta functionality
  • Feature flags - Individual feature toggles

Experimental Features

Beta features may be unstable. Enable at your own risk. Report bugs through the feedback option.
