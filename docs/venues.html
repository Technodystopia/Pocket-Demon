Venues

Managing Venues

Venues are the locations where concerts happen - from small clubs to massive festival grounds.

Creating a Venue

  1. Go to More → Venues
  2. Tap the + button
  3. Enter venue details:
    • Name - Venue name
    • City - City location
    • Country - Country (use picker for flags)
    • Address - Street address (optional)
    • GPS Coordinates - For map display
    • Capacity - Venue size (optional)
  4. Tap Save

Tip

Use the Enrich from Ticketmaster option to automatically fetch venue photos, address, and GPS coordinates.

Venue Navigation

PocketDemon uses progressive disclosure based on your collection size:

Collection Size Navigation Style
Less than 8 venues Direct list of all venues
8+ venues Grouped by city
12+ cities Grouped by country ‚Üí city

Venue Display Options

Customize how venues are shown in Settings ‚Üí Appearance:

  • Country Flags - Show colorful or grayscale flags
  • Compact Mode - Smaller list items
  • Sort Order - By name, visit count, or most recent

Cruise Ship Venues

For concerts on cruise ships:

  1. Create a venue with the ship name
  2. Enable Is Cruise Ship
  3. Events at this venue will be tracked as cruise concerts
  4. Add port stops with dates for itinerary tracking

Bucket List Venues

Track venues you want to visit but haven't yet:

  1. Create the venue
  2. Enable Bucket List toggle
  3. The venue appears in a separate "Bucket List" section

Bucket list features:

  • Venues are shown separately from visited venues
  • Excluded from venue statistics
  • Automatically removed when you attend an event there
  • Can manually mark as visited anytime

Venue Details

Tap a venue to see:

  • Photo - Venue image (from Ticketmaster or your photos)
  • Location - Address, city, country
  • Events - All events you've attended here
  • Statistics - Total visits, bands seen, first/last visit
  • Map - Location on map with directions button

Enriching Venue Data

PocketDemon can fetch venue information from Ticketmaster:

  1. Open venue details
  2. Tap Enrich
  3. Data fetched includes:
    • Official venue photo
    • Street address
    • GPS coordinates
    • Ticketmaster venue ID

Venue Map

View all your venues on a map:

  1. Go to More → Maps → Venue Map
  2. Venues are grouped by city with marker clustering
  3. Tap a marker to see venue name and visit count
  4. Tap again for directions

Getting Directions

Open navigation to any venue:

  1. Open venue details
  2. Tap the Directions button
  3. Opens in your default maps app (Google Maps, Apple Maps, etc.)

Note

Directions require GPS coordinates. Use the Enrich feature or manually enter coordinates.

Venue Achievements

Unlock badges by visiting venues:

  • Venue Visitor - Visit 10 different venues
  • Venue Explorer - Visit 25 different venues
  • Venue Hunter - Visit 50 different venues
  • Venue Collector - Visit 100 different venues
  • Venue Connoisseur - Visit 250 different venues

Regular visitor badges:

  • Regular - Visit the same venue 5 times
  • Local Hero - Visit the same venue 10 times
  • Home Turf - Visit the same venue 25 times
  • Resident - Visit the same venue 50 times
