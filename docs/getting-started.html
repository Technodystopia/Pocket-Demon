Getting Started

Creating Your Account

PocketDemon uses a local account system - no email required, no data sent to any server.

  1. Launch the app for the first time
  2. Tap Create Account
  3. Enter a nickname (this is how you'll be identified to buddies)
  4. Create a password (used to protect your local data)
  5. Tap Create

Tip

Your password protects access to your data on the device. If you forget it, you can restore from a backup, but local-only data will be lost.

Your First Concert Entry

Let's add your first concert to your collection.

Step 1: Create an Event

  1. Go to the Events tab
  2. Tap the + button
  3. Enter the event name (e.g., "Wacken Open Air 2024")
  4. Select or create a venue
  5. Set the date (and end date for multi-day events)
  6. Optionally add ticket price and notes
  7. Tap Save

Step 2: Add Bands to the Event

  1. Open the event you just created
  2. Tap Add Band
  3. Search for the band or create a new one
  4. Set your rating (1-5 stars)
  5. Add optional notes about the performance
  6. Repeat for all bands you saw

Note

Each band-at-event combination is called a "concert" in PocketDemon. One event can have many concerts.

Step 3: Enrich Your Data (Optional)

PocketDemon can fetch additional information about bands from online databases:

  1. Open a band's detail page
  2. Tap the Enrich button
  3. The app will fetch:
    • Country of origin, formation year
    • Genres and aliases
    • External links (Wikipedia, Discogs, etc.)
    • Artist photo and logo

Navigation Overview

PocketDemon has five main tabs at the bottom:

Home Quick actions, recent activity, "On This Day" memories
Bands Browse and manage all bands in your collection

Escape Go back

Setting Up Cloud Backup

While PocketDemon works 100% offline, you can optionally connect to Dropbox for:

  • Automatic cloud backup
  • Sync between devices
  • Photo backup
  • Buddy connections

To enable Dropbox sync:

  1. Go to Settings → Backup & Sync
  2. Tap Connect to Dropbox
  3. Authorize the app in your browser
  4. Enable Auto-Sync for automatic backups

Tip

Auto-sync runs when you leave the app, with a 15-minute cooldown to save battery.
