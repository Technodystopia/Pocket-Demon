# PocketDemon Documentation

This repository contains the user manual and documentation website for PocketDemon, a concert tracking app for music enthusiasts.

## About PocketDemon

PocketDemon helps you track your concert experiences, manage your music collection, and connect with fellow concert-goers. Key features include:

- **Event Tracking** - Log concerts with dates, venues, setlists, and photos
- **Band Management** - Track artists you've seen with detailed statistics
- **Achievements** - Unlock 53 badges across 7 categories
- **Buddy System** - Connect and share experiences with friends
- **Integrations** - Sync with Setlist.fm, Last.fm, Discogs, MusicBrainz, and more
- **Multi-platform** - Available on Android, iOS, macOS, Windows, and Linux
